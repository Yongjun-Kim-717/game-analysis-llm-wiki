# Game Page Schema

## Required Metadata

- `title`
- `page_type: game-analysis`
- `status`
- `game_slug`
- `genre`
- `platform`
- `evidence_level`
- `core_loop`
- `mechanics`
- `similar_games`
- `last_reviewed`

## Recommended Metadata

- `aliases`
- `original_query`
- `quality_score`
- `quality_level`
- `tags`

## Required Sections

- 기본 정보
- 한 줄 정의
- Core Loop
- 핵심 재미 가설
- 주요 시스템
- 장르 관점과 비교
- 차별점 분석
- 유사 게임 비교
- 근거 자료
- 검증 필요 사항
- 관련 Wiki Pages
- 유지보수 메모

## Recommended Sections

- 분석 품질
- 유저 반응 요약
- 플레이 경험 관찰

## Citation Rules

- Facts use source ids such as `[S1]`.
- Player reactions use review ids such as `[R1]`.
- Analysis-only statements use `[inference]`.
- Core loop items should include `[confidence: high|medium|low]`.
