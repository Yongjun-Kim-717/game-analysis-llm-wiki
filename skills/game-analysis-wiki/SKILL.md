# Game Analysis Wiki Skill

Use this skill when adding, reviewing, updating, validating, classifying, or comparing game analysis pages in this repository.

## Workflow

1. Read `AGENTS.md`, `RULES.md`, `SCHEMA.md`, and the latest `journal.md` entry.
2. Select the appropriate local skill in the Skill Runner.
3. Run the skill through `tools/skill-runner.js` or the GUI.
4. Confirm generated artifacts under `raw/`, `handoffs/`, `wiki/`, or `maintenance/`.
5. Validate pages with `schema.validate`.
6. Update or inspect `journal.md`.

## Available Local Skills

- Analyze New Game
- Compare Existing Games
- Update Existing Page
- Create Design Note
- Validate Wiki Page
- Tag & Classify Page
- Refresh Evidence
- Maintenance Sweep

## Comparison Axes

- genre similarity
- core loop similarity
- mechanic overlap
- progression structure
- randomness and player control
- fun factor
- differentiation
- reference value for the user's own game
