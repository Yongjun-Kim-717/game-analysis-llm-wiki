# Roguelike Deckbuilder

## 장르 정의

Roguelike Deckbuilder는 로그라이크식 run 반복과 덱빌딩 카드 게임의 선택 구조를 결합한 장르이다.

## 장르 관습

- run 단위 진행
- 무작위 또는 반무작위 보상
- 매 run마다 달라지는 빌드
- 실패 후 재도전
- 플레이어의 전략 학습 누적

## 주요 비교 축

| 비교 축 | 질문 |
| --- | --- |
| 덱 품질 관리 | 카드 추가와 제거가 얼마나 중요한가? |
| 경로 선택 | 위험과 보상이 명확히 충돌하는가? |
| 시너지 | 카드와 유물이 전략을 얼마나 바꾸는가? |
| 실패 학습 | 실패 원인을 판단 실수로 복기할 수 있는가? |
