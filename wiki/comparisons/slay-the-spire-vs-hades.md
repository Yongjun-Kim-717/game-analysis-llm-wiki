# Slay the Spire vs Hades

## 비교 목적

두 게임은 모두 run 기반 반복과 build 변주성을 가진다. 그러나 `Slay the Spire`는 턴제 카드 선택과 덱 품질 관리가 중심이고, `Hades`는 실시간 액션 전투와 실패의 서사화가 중심이다.

## 비교표

| 비교 축 | Slay the Spire | Hades |
| --- | --- | --- |
| 핵심 행동 | 카드 선택과 전투 계산 | 이동, 공격, 회피, boon 선택 |
| Core Loop | 경로 선택 -> 카드 전투 -> 보상 선택 -> 덱 조정 | 방 전투 -> boon 선택 -> 보스 도전 -> 사망 후 귀환 |
| 실패 후 동기 | 전략 복기와 다음 run 개선 | NPC 대화, 관계, 서사 진행 |
| 차별적 재미 | 덱을 오염시키지 않는 판단 | 실패가 이야기 진행으로 보상됨 |

## 결론

`Slay the Spire`의 차별점은 액션 숙련이 아니라 **선택의 누적과 덱 품질 관리**에 있다. 반면 `Hades`는 실패 이후에도 감정적/서사적 보상이 이어지는 구조가 강하다.
